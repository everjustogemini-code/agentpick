Built BYOK support for AgentPick router/dashboard.

- Added encrypted per-account BYOK storage on `DeveloperAccount.byokKeys`.
- Added `/api/v1/router/keys` CRUD for saved provider keys plus a BYOK savings summary.
- Router now prefers saved user keys before platform keys and skips platform vault usage tracking when BYOK is injected.
- Router call records now persist real `costUsd` metadata, enabling BYOK savings reporting.
- Main `/dashboard` now includes a BYOK panel for saving, pausing, rotating, and deleting keys.
- Verified with `npx vitest run src/__tests__/router.test.ts src/__tests__/byok.test.ts`, `npm run build`, and `npx prisma db push`.
